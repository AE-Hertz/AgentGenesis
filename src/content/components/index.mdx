This is **not** an npm library. Instead, it's a collection of reusable code snippets designed to accelerate your AI development process.

## Meaning

AgentGenesis is not distributed via npm and does not need to be installed as a dependency. 
Simply pick the code you need, copy and paste it into your project, and build on top of it according to your requirements. 
The code is entirely yours to use and modify.

Use AgentGenesis as a reference to construct your own AI flows.
